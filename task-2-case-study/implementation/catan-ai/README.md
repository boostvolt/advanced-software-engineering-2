# catan-ai

Claude Opus generated implementation of *Siedler von Catan* for the ASE2 Team 08 case study. Used as the AI side of the human-vs-AI comparison; the human-authored counterpart is at [`../catan-human/`](../catan-human/).

## Provenance

- **Generated by:** Claude Opus, in a single agentic session of about 10 minutes wall-clock time.
- **Prompt:** [`../PROMPT.md`](../PROMPT.md) — pasted verbatim as the only message; no follow-up prompts.
- **Specification:** [`../SPEC.md`](../SPEC.md) — consolidated English version of `../spec/PM1 Projekt 3 Anleitung.pdf` and `../spec/PM1 Projekt 3 Kick-Off.pdf`. The PDFs are authoritative; SPEC.md only consolidates.
- **Reflection:** [`GENERATION_LOG.md`](GENERATION_LOG.md) — final file list, build status, design decisions, self-flagged uncertainties.
- **Fixed library:** the `ch.zhaw.hexboard` package is the read-only scaffold from the original assignment, identical in `catan-human/` and `catan-ai/`.

## Build and run

```bash
mvn compile
mvn test
mvn exec:java -Dexec.mainClass=ch.zhaw.catan.App
```

`mvn test` runs 8 tests (3 required `requirement*` tests + 5 additional). All pass at the time `GENERATION_LOG.md` was written.
